CSS radi na Box modelu. Tipicna web stranica sastoji se od vise "kutija" spojenih zajedno od vrha prema kraju. CSS Box model je "kutija" koja omotava oko svakog HTML elementa. Sastoji se od:

  • Margina - Cisti prostor izvan Bordera, transparenta je
  • Bordera - Ide okolo paddinga i contenta
  • Paddinga - Cisti prostor oko Contenta, transparentan je
  • Contenta - sadrzaj kutije, gdje se tekst i slike pokazuju
